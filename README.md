# 🛍️ Shopping Web App (Next.js)

A learning-focused e-commerce web application built using **React, Next.js, and modern frontend practices**.

This project was created to understand:
- Real-world Next.js project structure
- Component-based UI development
- Global state management using React Context
- Proper Git & GitHub workflow from local setup to push

## ▶️ Run Locally

```bash
git clone https://github.com/Vignesh-P-C/shopping-web-app
cd shopping-web-app
pnpm install
pnpm dev
```
open: http://localhost:3000
